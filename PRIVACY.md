# Privacy

Vantadeck is local-first. No account, telemetry, analytics, advertising, or remote
service is required. Core scanning, projects, launching, VCS, and health checks run
without network access.

Update checks and the Tools Hub are separate opt-in settings. Their user interfaces
must identify the destination and last request time. Vantadeck does not sell or share
usage data. Project paths and source-control metadata remain on the local machine.
